Mike Tysons Punch Out Speedrunning

Guide based on Sinisterwon's tutorial videos.

Minor Circuit

Glass Joe: (:42)

  • Up+B x3, then B
  • Up+B x3
  • B x15
  • Up+B
  • B, A, A, B

Von Kaiser: (:37 - :39)

  • Phase 1: Up+B, Left-Up, Up+B x7, Left-Up, Up+B, Start
  • Phase 2: Wait for gloves to drop, Tap-Up, B, Hold B, Start
  • Phase 3 (w/Star): After flash, Left-Up, Start
  • Phase 3 (w/Star): Hold Left, Tap up as soon as Little Mac starts moving, 2xtap down, Hold Start
  • Phase 3 (w/o Star): Tap-Up, B, Hold B, Start

Piston Honda (:51)

  • Phase 1: Hold-Up, Let go of Up, B, B, B
  • Hold-Up (Press Start, pause for Honda's face to go back to normal, Start, Start)
  • Let go of up. Tap (don't buffer) Up+B x2, Dodge, Up+B x3, Start
  • Phase 2: Hold Start
  • Phase 3: Hold-Up, Let go of Up, B, B, B
  • Hold-Up (Press Start, pause for Honda's face to go back to normal, Start, Start)

Major Circuit

Don Flamenco (:15)

  • Phase 1: Hold B+Left, release when dodge starts, press Left again, Up+B, buffer Start.
  • Phase 2: After flash, quick dodge left, Up+B x5

King Hippo (:)

  • Very RNG based.
  • Buffer Strat
  • Phase 1: Buffer three right dodges. Counter his punches with face punches. When his stomach is open, punch his belly repeatedly. Repeat.

Great Tiger ()

  • Phase 1: Buffer Up+B, dodge, delay A, repeat for three stars, buffer Start x2
  • Up+B, dodge, delay A, then Up+B when gem flashes.
  • When his legs move three times, Start
  • Phase 2: Hold Start
  • Phase 3: Hold Start

Bald Bull ()

  • Phase 2 (first punch star): Buffer Up+B, tap Up+B, Start,
  • Phase 2 (no first punch star): Buffer Up+B, tap Up+B,
  • Phase 3:
